Description

We build the project with Dash from Plotly

We used a dataset from kaagle with the rank of each player per country and the results from each different game. In this project we are vizualizing the data according to the selection of the user with a more interactive and efficient to understand way than a simple document. In this project we want to highlight the functionality of the plotly-dash. With this kind of developement we could make applications in zero time and share it in our local company network for many users or to share it on internet with Technologies as Amazon Web Services, Azure etc. For more details you are welcome to deep into code or run the Docker image easily at your prefferable machine
